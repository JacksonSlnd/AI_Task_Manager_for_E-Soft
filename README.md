# Менеджер задач (Task Manager)

Веб-приложение для управления задачами и встроенный инструмент для администрирования (экспорт данных).

## Стек технологий
* **Frontend:** React, Vite
* **Backend:** Node.js, Express, PostgreSQL
* **Скрипты:** Python 3 (для выгрузки данных)

---

## Быстрый запуск проекта

> **Важно:** Для полноценной работы приложения бэкенд и фронтенд должны быть запущены **одновременно в двух разных окнах терминала**.

### 1. Подготовка окружения и базы данных
1. Убедитесь, что у вас установлен **PostgreSQL** и запущена его служба.
2. Создайте базу данных (например, `task_manager`).
3. В корне папки `nodejsServ` найдите файл `.env.example`. 
4. Создайте его копию и назовите её просто **`.env`**.
5. Заполните созданный файл `.env` своими реальными данными подключения:

```env
DB_USER=ваш_пользователь_postgres
DB_PASSWORD=ваш_пароль
DB_HOST=localhost
DB_PORT=5432
DB_DATABASE=task_manager
```

### 2. Запуск Бэкенда (Терминал №1)
1. Откройте первое окно терминала и перейдите в папку с сервером: `cd nodejsServ`
2. Установите зависимости и запустите сервер:
* Установка зависимостей сервера: `npm install`
* Запуск сервера в режиме разработки: `npm run dev`


### 3. Запуск Фронтенда (Терминал №2)
1. Откройте второе (новое) окно терминала и перейдите в папку клиента: `cd reactApp`
2. Установите зависимости и запустите интерфейс:
* Установка зависимостей фронтенда: `npm install`
* Запуск клиентской части (Vite): `npm run dev`
3. Перейдите по адресу, который выдаст Vite в консоли (обычно http://localhost:5173), чтобы открыть календарь задач.


### Использование Python-скрипта для экспорта задач

В проекте присутствует административный скрипт `export_tasks.py`, предназначенный для быстрой выгрузки всех существующих задач из базы данных PostgreSQL в файл формата `.csv`.

Скрипт запускается из папки `nodejsServ` и автоматически использует настройки подключения из вашего файла `.env`.

#### Инструкция по запуску скрипта:
1. Находясь в папке nodejsServ, создайте и активируйте виртуальное окружение Python:
* Создание окружения: `python -m venv .venv`
* Активация в Windows (PowerShell): `.venv\Scripts\Activate.ps1`
2. Установите необходимые зависимости для работы с БД: `pip install -r requirements.txt`
3. Запустите скрипт выгрузки: `python export_tasks.py`

После успешного выполнения в корне папки появится файл exported_tasks.csv, оптимизированный для открытия в Microsoft Excel (с кодировкой UTF-8 и разделителем ;).
